Cài đặt uv rồi chạy tiếp (Khuyên dùng)
Bạn có thể làm theo gợi ý của Terminal bằng cách cài đặt uv qua snap, hoặc dùng lệnh cài đặt chính thức. Hãy chạy lệnh sau:

bash
sudo snap install astral-uv
(Bạn sẽ cần nhập mật khẩu máy tính)

Sau khi cài đặt xong uv, bạn chạy lại các lệnh lúc nãy:

bash
uv python install 3.11
uv sync --frozen
sh webui.sh

=====
Chạy chương trình:

.\webui.bat
